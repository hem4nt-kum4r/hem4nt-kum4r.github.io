Redis Cluster Host View

Managing and expanding a Redis cluster can be confusing, especially when replicas are identified only by IP addresses. To simplify this, a browser-based tool was developed that visually maps masters and replicas, shows their locations, and tracks their status in real time. It generates the cluster topology as a Mermaid flowchart and runs entirely client-side without sending data externally.
